# Inventory Item Image Prompts

## Style Guide
- **Format**: Square 1:1 aspect ratio, 500x500 pixels, PNG
- **Background**: Clean white background (transparent if possible)
- **Angle**: Side profile with slight 3/4 angle, or diagonal composition for long items
- **Framing**: Object centered in frame with padding
- **Style**: Realistic 3D render, product photography, high detail, game inventory icon
- **Rules**: No text, no watermarks, no extra objects
- **Tool**: Nano Banana (or any AI image generator)

## Template
```
Square 1:1 aspect ratio, 500x500 pixels. Realistic 3D render of [ITEM DESCRIPTION], [ANGLE/COMPOSITION], centered in frame, clean white background, product photography, no text, high detail, game inventory icon
```
